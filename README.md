# pwntools_env

## 介绍
自动部署pwn题环境
pwntools gdb ROPgadget 
gdb插件来自gitee，有效避免github连接不上的问题

## 安装教程
```sh
git clone https://gitee.com/liweijun0302/pwntools_env.git
cd pwntools_env
./pwntools_env.sh
```

## 使用说明

1.  安装gdb插件时默认安装peda，如果需要安装pwndbg，在pwntools_env.sh 第85行有注释教学
